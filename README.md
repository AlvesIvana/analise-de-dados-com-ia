# 📖 Miniguia de Estudos

## O que é análise de dados?

A análise de dados é o processo de coletar, organizar, interpretar e transformar dados em informações úteis para auxiliar na tomada de decisões.

Empresas utilizam análise de dados para identificar padrões, prever tendências e melhorar resultados estratégicos.

---

## Como a IA auxilia na análise de dados?

A Inteligência Artificial potencializa a análise de dados através de:

- Automação de processos;
- Identificação de padrões;
- Geração de previsões;
- Criação de insights;
- Processamento de grandes volumes de dados;
- Apoio à tomada de decisão.

Ferramentas modernas utilizam IA para acelerar análises e tornar os processos mais eficientes.

## Ferramentas importantes

### Power BI
Ferramenta da Microsoft utilizada para criação de dashboards e visualização de dados.

### Tableau
Plataforma utilizada para análise e visualização interativa de informações.

### NotebookLM
Ferramenta de IA utilizada para organização de estudos, pesquisas e geração de resumos.

### Amazon SageMaker
Plataforma da AWS utilizada para desenvolvimento de modelos de Machine Learning.

### Gemini
Assistente de IA da Google utilizado para produtividade e automação.

# 📚 Glossário

## Inteligência Artificial (IA)
Tecnologia capaz de simular capacidades humanas utilizando algoritmos e dados.

## Dashboard
Painel visual utilizado para acompanhar indicadores e métricas.

## Insight
Informação relevante obtida através da análise de dados.

## Machine Learning
Área da IA responsável pelo aprendizado baseado em dados.

## Business Intelligence (BI)
Conjunto de processos e ferramentas utilizadas para análise estratégica de dados.

## Automação
Execução automática de tarefas e processos.

## Mineração de Dados
Processo de descoberta de padrões em grandes conjuntos de dados.

## Análise Preditiva
Uso de dados e algoritmos para prever resultados futuros.
